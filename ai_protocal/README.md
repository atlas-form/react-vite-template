# ai_protocal

本目录用于约束 AI 在本项目中的开发行为。

## 文档索引

- `AI_CODING_PROTOCOL.md`
  - 说明 AI 如何按本项目框架写代码（流程、规则、约束、自检）。
- `DIRECTORY_GUIDE.md`
  - 说明当前项目每个主要目录的职责与放置规则。
- `TODO_PROGRESS_PROTOCOL.md`
  - 说明 AI 必须先列 Todo、完成即划掉、并在中断后按 Todo 恢复。

## 建议使用顺序

1. 先读 `AI_CODING_PROTOCOL.md`（明确怎么写）。
2. 再读 `DIRECTORY_GUIDE.md`（明确写到哪里）。
3. 最后读 `TODO_PROGRESS_PROTOCOL.md`（明确如何跟踪和恢复进度）。
