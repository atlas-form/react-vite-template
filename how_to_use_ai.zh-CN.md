# How To Use AI（如何使用 AI）

## 最简用法
你只需要一句话：

```text
先阅读 ai_protocal 目录，然后按规范实现：<你的需求>
```

## AI 执行前置规则（必须）
AI 在开始改代码前必须先做这三件事：

1. 先阅读 `ai_protocal/` 下全部协议文件。
2. 先输出 Todo List（Markdown checklist）。
3. 每完成一项就勾选并划掉，持续更新进度。

对应协议文件：
- `ai_protocal/AI_CODING_PROTOCOL.md`
- `ai_protocal/DIRECTORY_GUIDE.md`
- `ai_protocal/TODO_PROGRESS_PROTOCOL.md`

## 为什么这样做
- 减少你输入成本：不需要再写很多上下文。
- 降低中断损失：网络断开后可按 Todo 继续。
- 保持一致性：AI 统一按项目协议实现。

## 推荐提需求格式

```text
先阅读 ai_protocal 目录。
先列 Todo list，并在完成后逐条划掉。
然后实现：<功能需求>
```
